🎓 Placement Tracker

A simple and efficient Placement Tracker application designed to help students manage and track placement opportunities in one place. The application allows users to keep records of companies, job roles, application status, and placement updates through a clean and user-friendly interface.

✨ Features

✔️ Track placement applications
✔️ Manage company details and job roles
✔️ Update application status easily
✔️ Simple and responsive UI
✔️ Easy navigation and fast performance

🛠️ Technologies Used
🌐 HTML
🎨 CSS
⚡ JavaScript
🟢 Node.js
🚀 Express.js
🗄️ MySQL

🎯 Usage

1️⃣ Add company and job details
2️⃣ Track application status
3️⃣ Manage placement records
4️⃣ View placement updates easily
